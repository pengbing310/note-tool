# GitHub静态备忘录工具

一个完全基于GitHub Pages的静态备忘录工具，无需服务器，无需登录，数据存储在GitHub仓库。

## 功能特点

- ✅ **无需登录** - 打开即用
- ✅ **GitHub存储** - 数据安全存储在GitHub仓库
- ✅ **密码保护文件夹** - 支持公开/私密文件夹
- ✅ **响应式设计** - 完美支持电脑和手机
- ✅ **实时保存** - 修改自动保存
- ✅ **删除功能** - 支持文件夹和备忘录删除
- ✅ **数据导出** - 支持JSON格式导出
- ✅ **中文支持** - 完全支持中文字符

## 快速开始

### 方法一：使用GitHub Pages（推荐）

1. **Fork这个仓库**到你的GitHub账户
2. **开启GitHub Pages**：
   - 进入仓库设置 → Pages
   - 选择分支（main）和目录（/root）
   - 保存后访问：`https://你的用户名.github.io/仓库名`

### 方法二：本地使用

1. 下载所有文件到本地
2. 用浏览器打开 `config.html` 进行配置
3. 开始使用

## 配置说明

首次使用需要配置GitHub信息：

1. **GitHub用户名** - 你的GitHub账户名
2. **仓库名称** - 存储备忘录数据的仓库（建议：memo-data）
3. **Personal Access Token** - 需要 `repo` 权限

### 如何获取Token

1. 登录GitHub，访问：https://github.com/settings/tokens
2. 点击 "Generate new token"
3. 输入名称（如：memo-app）
4. 选择 `repo` 权限
5. 点击 "Generate token"
6. **复制生成的Token**（只会显示一次）

## 使用说明

### 创建文件夹
- 点击"新建文件夹"按钮
- 输入文件夹名称
- 选择权限：公开（所有人可见）或私密（需要密码）
- 设置密码（私密文件夹需要）

### 创建备忘录
1. 选择文件夹
2. 点击"新建备忘录"
3. 输入标题和内容
4. 自动保存到GitHub

### 数据管理
- **导出数据**：点击底部"导出数据"按钮
- **重新配置**：点击右上角配置图标
- **删除功能**：支持删除文件夹和备忘录

## 技术特点

- **纯静态**：仅使用HTML/CSS/JavaScript
- **无服务器**：完全依赖GitHub Pages
- **安全存储**：Token本地加密，不上传服务器
- **响应式**：完美适配各种屏幕尺寸
- **离线备份**：浏览器本地存储备份

## 隐私说明

- 数据存储在**你的GitHub仓库**，完全由你控制
- 私密文件夹内容在本地加密后才上传
- Token仅保存在浏览器本地，不清除数据不会丢失

## 注意事项

1. 建议使用**私有仓库**存储敏感数据
2. 定期导出数据备份
3. Token泄露请立即在GitHub撤销
4. 清除浏览器数据会丢失本地配置

## 故障排除

### 无法连接GitHub
- 检查Token是否正确
- 确认Token有 `repo` 权限
- 确认仓库存在且有访问权限

### 数据不同步
- 检查网络连接
- 确认仓库名正确
- 尝试重新配置
