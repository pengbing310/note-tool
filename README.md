# GitHub静态备忘录工具 - URL参数分享配置版

一个完全基于GitHub Pages的静态备忘录工具，支持通过URL参数快速分享配置到其他设备。

## 🚀 主要特性

### 核心功能
- ✅ **无需登录** - 打开即用
- ✅ **GitHub存储** - 数据安全存储在GitHub仓库
- ✅ **密码保护文件夹** - 支持公开/私密文件夹
- ✅ **响应式设计** - 完美支持电脑和手机
- ✅ **实时保存** - 修改自动保存
- ✅ **跨设备同步** - 通过URL分享配置

### 新增功能
- 🔗 **URL参数分享配置** - 一键生成分享链接
- 📱 **多设备快速配置** - 打开链接自动完成配置
- 🔒 **安全加密** - Token本地加密存储
- 📤 **数据导出** - 支持JSON格式导出

## 🎯 快速开始

### 方法一：使用GitHub Pages（推荐）
1. **Fork这个仓库**到你的GitHub账户
2. **开启GitHub Pages**：
   - 进入仓库设置 → Pages
   - 选择分支（main）和目录（/root）
   - 保存后访问：`https://你的用户名.github.io/仓库名`

### 方法二：本地使用
1. 下载所有文件到本地
2. 用浏览器打开 `index.html`
3. 首次使用会自动跳转到配置页面

## ⚙️ 配置说明

### 首次配置
1. 打开页面 → 自动跳转到配置页面
2. 填写GitHub信息或选择本地存储
3. 点击"保存配置"

### 获取GitHub Token
1. 登录GitHub，访问：https://github.com/settings/tokens
2. 点击 "Generate new token"
3. 输入名称（如：memo-app）
4. 选择 `repo` 权限
5. 点击 "Generate token"
6. **复制生成的Token**（只会显示一次）

## 🔗 URL参数分享配置

### 生成分享链接
1. 在备忘录主页面
2. 点击右上角"分享配置"按钮
3. 复制生成的链接

### 在其他设备使用
1. 在其他设备的浏览器中打开分享链接
2. 系统会自动导入配置
3. 无需重新输入GitHub信息
